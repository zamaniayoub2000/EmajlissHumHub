# PDF viewer

Easily access, view, and navigate through your documents with our user-friendly PDF Viewer module. Open PDF files directly in your browser without the need to download them.

## Key Features
- **Presentation Mode:** Fullscreen document viewing ideal for smooth slideshows or document review.
- **Sidebar Navigation:** Quickly browse through pages with a convenient sidebar overview.
- **Document Rotation:** Easily adjust document orientation for optimal viewing.
- **Support for Odd Spreads:** Ensures seamless viewing of all document layouts, including odd spreads.
