# Homepage

**Create custom homepages for registered users, guests, and members of specific groups.**

## Features

- Homepages can be different for registered users, guests and members of specific groups.
- Pages can be created from a Rich text editor, HTML editor, iframe or redirect to an internal URL
- Select which widgets to display
- Select the display layer (full width, widgets on the right, etc.)
- Special tags that are automatically replaced (e.g., `{userFirstname}`) (for Rich text editor and HTML editor)

Can be used with the [Menu Manager module](https://marketplace.humhub.com/module/menu-manager) to add a "Home" item on the top menu.

If the Custom Pages module is enabled, the HTML editor will be the TinyMCE Rich Text Editor instead of a simple source code editor.
